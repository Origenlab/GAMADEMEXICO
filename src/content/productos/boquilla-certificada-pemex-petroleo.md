La **boquilla PEMEX Series PX-200** está específicamente diseñada y certificada para cumplir con las **Normas de Referencia PEMEX (NRF)** aplicables a equipos de combate de incendios en instalaciones petroleras mexicanas. Fabricada en bronce fundido, incluye toda la documentación requerida para auditorías.

## Especificaciones Técnicas

| Característica | Especificación |
|----------------|----------------|
| Marca | Nacional (Fabricación Mexicana) |
| Modelo | PEMEX Series PX-200 |
| Rango de Flujo | 95 - 125 - 150 - 200 GPM (360 - 475 - 570 - 757 LPM) |
| Presión de Trabajo | 100 PSI (7 BAR) |
| Entrada | 1.5" NST hembra con giro libre |
| Material Cuerpo | Bronce fundido libre de porosidades |
| Selector de Flujo | 4 posiciones con indexado positivo |
| Patrones de Descarga | Chorro sólido a niebla 90° |
| Peso | 2.2 kg (4.8 lbs) |
| Certificaciones | NRF-PEMEX, NFPA 1964, API |

## Características Destacadas

- **Cumple NRF-PEMEX** para instalaciones petroleras
- **Fabricación 100% mexicana** con soporte local
- **Bronce fundido** libre de porosidades
- **4 posiciones de flujo** indexadas
- **Documentación PEMEX completa** incluida
- **Compatible con espuma AFFF** para hidrocarburos

## Resistencia a Ambiente Petrolero

El bronce fundido de la PX-200 resiste:

- Exposición a vapores de hidrocarburos
- Ambiente con H2S (ácido sulfhídrico)
- Condiciones costeras (instalaciones offshore)
- Altas temperaturas ambientales
- Lluvia y humedad extrema
