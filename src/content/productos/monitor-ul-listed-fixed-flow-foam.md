El **Monitor UL Listed Fixed Flow** es un equipo de alto rendimiento certificado bajo **UL Standard 162** para sistemas de agua y espuma. Fabricado en **acero inoxidable** (SS304, SS316, SS316L), está diseñado para instalaciones fijas donde se requieren **altos caudales** y **chorros de largo alcance**. Su sistema de **inducción de espuma UL Listed** lo hace ideal para protección contra incendios de líquidos inflamables.

## Especificaciones Técnicas

| Característica | Especificación |
|----------------|----------------|
| Certificación | **UL Listed** bajo UL Standard 162 |
| Material | Acero Inoxidable SS304 / SS316 / SS316L |
| Tipo | Instalación fija con entrada bridada |
| Boquillas | Flujo fijo y flujo variable UL Listed |
| Sistema de Espuma | Inductor con válvula dosificadora |
| Aplicación | Alto caudal y largo alcance |
| Compatibilidad | Jet-Ratio controller sin modificación |

## Sistema de Inducción de Espuma

El monitor incluye un **mecanismo de inducción de espuma certificado UL Listed** con:

- **Válvula dosificadora (Metering Valve)** para control preciso
- Inducción proporcional al flujo de agua
- Compatible con concentrados AFFF y AR-AFFF
- **Boquilla Aqua Foam no aspirante** incluida
- Compatible con **Jet-Ratio controller** sin modificaciones
