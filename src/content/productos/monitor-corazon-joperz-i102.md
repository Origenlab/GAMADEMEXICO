El monitor contra incendio Joperz modelo I-102 tipo corazón de doble volante es un equipo de alto rendimiento diseñado para aplicaciones industriales con resistencia a la intemperie, incluyendo ambientes de clima marítimo con exposición a agua salada. Con un flujo de hasta 1,000 GPM, ofrece cobertura excepcional para protección contra incendio en instalaciones de gran escala.

## Especificaciones Técnicas

| Característica | Especificación |
|----------------|----------------|
| Material | Bronce |
| Giro Horizontal | 360° continuo (volante) |
| Giro Vertical | 120° (volante) |
| Entrada | Brida 4" ANSI 150# R.F. |
| Salida | 2.5" macho rosca NST |
| Flujo Máximo | 1,000 GPM (3,785 LPM) |
| Rango de Flujo | 250 - 1,000 GPM |
| Presión de Operación | 100 PSI (7 bar) |
| Lubricación | Graseras para lubricación |
| Acabado | Pintura roja anticorrosiva |
| Peso | 88.18 lbs (40 kg) |
| Fabricación | Nacional bajo normas ASTM, NFPA, PEMEX |

## Características Destacadas

- Alto flujo de hasta 1,000 GPM para descarga masiva
- Resistencia comprobada a clima marítimo y agua salada
- Doble volante para control preciso en ambas direcciones
- Fabricado bajo normas ASTM, NFPA y PEMEX
- Peso robusto de 40 kg para estabilidad en operación
- Compatible con boquillas reguladoras y eductoras de espuma

## Aplicaciones

- Plantas petroquímicas y refinerías
- Plataformas offshore y marítimas
- Terminales marítimas y portuarias
- Instalaciones industriales de alto riesgo
- Plantas de generación de energía

## Resistencia Marítima

El Joperz I-102 está especialmente diseñado para **ambientes marítimos agresivos**, con resistencia comprobada al agua salada y condiciones de humedad extrema, haciéndolo ideal para instalaciones costeras y plataformas offshore.
